# 鱿郁仔仔 UI设计规范

> 版本：v1.0 | 更新日期：2026-05-23

---

## 一、设计原则

### 1.1 基础版：极简纯净
- 仅保留核心功能入口
- 无灰色不可用按钮 — VIP功能不显示就是不存在
- 唯一的例外：一个「升级VIP」按钮，引导用户了解进阶功能

### 1.2 操作效率优先
- 核心操作（搜索、安装）2步内完成
- 常用设置不超过3层嵌套
- 状态信息一目了然（顶部状态栏 + 任务列表）

---

## 二、配色方案

### 2.1 主色调 — 深色主题

| 用途 | 色值 | 说明 |
|---|---|---|
| 主背景 | #1A1A2E | 深蓝黑，类似Switch OLED的暗色调 |
| 次背景/卡片 | #16213E | 略浅一层 |
| 主强调色 | #E94560 | 霓虹红粉，用于主按钮、激活状态 |
| 次强调色 | #0F3460 | 深蓝，用于次要操作 |
| 成功 | #00C853 | 绿色，安装成功状态 |
| 警告 | #FFD600 | 黄色，需注意状态 |
| 错误 | #FF1744 | 红色，失败状态 |
| 文字主色 | #E0E0E0 | 浅灰白 |
| 文字次色 | #9E9E9E | 灰色辅助文字 |
| 分割线 | #2A2A4A | 深色分割 |

### 2.2 VIP专属色
- VIP按钮：金色渐变 #FFD700 → #FFA000
- VIP标识：金色徽章 + 皇冠图标

## 三、组件规范

### 3.1 窗口结构

```
┌──────────────────────────────────────────┐
│  顶部状态栏 (设备状态 | 版本 | VIP状态)     │
├──────────────────────────────────────────┤
│                                          │
│              主内容区                      │
│      (搜索 → 结果 → 安装队列)              │
│                                          │
├──────────────────────────────────────────┤
│  底部操作栏 (设置 | 本地安装 | 升级VIP)     │
└──────────────────────────────────────────┘
```

### 3.2 搜索输入框
- 居顶，大号（高度 48px）
- placeholder："输入游戏名称..."
- 右侧带搜索图标按钮
- 输入时实时调用后端API，500ms防抖
- 下拉显示搜索结果（游戏封面缩略图 + 标题 + 版本 + 大小）

### 3.3 结果列表
- 每项显示：游戏封面图（64x64）+ 标题 + 版本号 + 文件大小
- 已安装的游戏显示勾标记
- 选中后展开详情：本体/更新/DLC 各自文件大小

### 3.4 安装按钮
- 主按钮：霓虹红粉 (#E94560)，圆角 8px，文字白色
- 宽 200px，高 44px
- 点击后变为「安装中...」禁用态 + 进度条

### 3.5 进度条
- 高度 6px，背景 #2A2A4A，填充色 #E94560
- 下方显示：当前文件名称 + 已传/总大小 + 速度 + ETA（VIP）

### 3.6 状态栏（顶部常驻）
- 左侧：Switch连接状态（绿色圆点 = 已连接，红色 = 未连接）
- 中间：当前操作状态文字
- 右侧：VIP状态（金色徽章 = 已激活，灰色 = 基础版）+ 版本号

### 3.7 设置面板
- 侧滑或弹窗形式
- 分组：网盘设置 / 通知设置 / 通用设置
- 每个设置项：标签 + 输入框/开关，操作即保存

### 3.8 VIP升级页
- 独立路由页面
- 顶部大标题："解锁进阶功能"
- 左右对比卡片：基础版 vs VIP版功能列表
- 底部激活码输入框 + 激活按钮
- 小字提示："兑换码可闲鱼搜索「鱿郁仔仔」"

## 四、字体与间距

### 4.1 字体
- 系统默认中文字体（Microsoft YaHei / PingFang SC）
- 等宽字体用于路径/代码显示：Consolas
- 标题字号：20px / 16px / 14px
- 正文字号：14px
- 辅助文字：12px

### 4.2 间距基准
- 基础单位：8px
- 组件内边距：16px (2x)
- 组件间距：24px (3x)
- 页面边距：32px (4x)

---

## 五、动画与反馈

### 5.1 过渡动画
- 页面切换：淡入淡出 200ms ease
- 列表项展开/收起：高度过渡 300ms ease
- 按钮 hover：背景色过渡 150ms

### 5.2 加载状态
- 骨架屏：浅灰色脉动条（搜索结果加载中）
- 进度条：平滑递增动画

### 5.3 空状态
- 搜索无结果：显示友好图标 + "未找到匹配游戏，试试其他关键词"
- 设备未连接：大号警告图标 + 连接指引文字 + 重新检测按钮
