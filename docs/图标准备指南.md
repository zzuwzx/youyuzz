# 图标文件准备指南

鱿郁仔仔需要以下图标文件用于打包：

## 必需文件

1. **`pc-client/frontend/assets/icon.ico`**
   - Electron 应用图标
   - 用于安装程序和桌面快捷方式

2. **`pc-client/backend/icon.ico`**
   - Python 后端可执行文件图标
   - 用于 backend.exe

## 图标规格

- **格式**: ICO（Windows 图标格式）
- **尺寸**: 包含以下尺寸的图标
  - 16x16 像素
  - 32x32 像素
  - 48x48 像素
  - 256x256 像素（推荐）
- **颜色深度**: 32 位（支持透明度）
- **文件大小**: 建议 < 500KB

## 制作方法

### 方法 1: 在线转换工具

1. 准备一个 256x256 的 PNG 图片（带透明背景）
2. 访问在线转换工具：
   - https://convertio.co/png-ico/
   - https://icoconvert.com/
3. 上传 PNG 文件
4. 选择多尺寸输出（16, 32, 48, 256）
5. 下载 ICO 文件
6. 复制到上述两个位置

### 方法 2: ImageMagick（命令行）

```powershell
# 安装 ImageMagick
winget install ImageMagick.ImageMagick

# 转换 PNG 到 ICO（包含多尺寸）
magick convert logo.png `
  -resize 256x256 `
  -define icon:auto-resize=256,48,32,16 `
  icon.ico

# 复制到目标位置
Copy-Item icon.ico "pc-client\frontend\assets\icon.ico"
Copy-Item icon.ico "pc-client\backend\icon.ico"
```

### 方法 3: Greenfish Icon Editor Pro（免费软件）

1. 下载: https://greenfishsoftware.blogspot.com/p/greenfish-icon-editor-pro.html
2. 打开 PNG 图片
3. 使用 "Add Image" 添加不同尺寸版本
4. 保存为 ICO 格式

### 方法 4: Photoshop + ICO 插件

1. 安装 ICO 格式插件
2. 创建多尺寸图层
3. 导出为 ICO 格式

## 临时方案

如果没有图标文件，可以暂时使用系统默认图标：

1. 修改 `pc-client/backend/build.spec`:
```python
exe = EXE(
    # ... 其他配置 ...
    icon=None,  # 使用默认图标
)
```

2. 修改 `pc-client/frontend/electron-builder.yml`:
```yaml
win:
  icon: null  # 使用默认图标
```

注意：使用默认图标可能会影响用户体验，建议尽快添加自定义图标。

## 验证图标

添加图标后，验证文件存在：

```powershell
Test-Path "pc-client\frontend\assets\icon.ico"  # 应返回 True
Test-Path "pc-client\backend\icon.ico"          # 应返回 True
```

## 设计建议

- 使用简洁、易识别的设计
- 在小尺寸（16x16）下仍能辨识
- 使用项目的品牌颜色
- 避免过多细节
- 可以使用游戏相关的元素（如 Switch、游戏手柄等）
