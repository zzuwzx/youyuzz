# 鱿郁仔仔 注意事项

> 版本：v1.0 | 更新日期：2026-05-23

## 一、开发禁忌 (DO NOT)

### 1.1 MTP传输
- **禁止并行传输**：MTP协议不支持多文件并发写入Switch，必须严格串行。并行会导致文件损坏或Switch DBI崩溃。
- **禁止在传输中断开USB**：断开前必须调用 PyMTP 的 `disconnect()`，否则Switch可能卡在DBI模式下需重启。
- **禁止传输超大单文件到TF卡**：FAT32文件系统单文件上限4GB，超过需自动切换到 exFAT 分区或 nand。

### 1.2 Switch 文件系统
- **禁止删除 nintendo/ 和 emummc/ 目录下的任何文件**：这是大气层虚拟系统的核心数据，误删会导致系统崩溃。
- **大气层升级时不要格式化TF卡**：只删除除 nintendo/ 和 emummc/ 之外的旧大气层文件。
- **TF卡路径区分大小写**：虽然FAT32不区分，但某些homebrew软件会检查大小写。

### 1.3 网盘操作
- **不要频繁请求网盘API**：夸克/百度/阿里都有频率限制，一旦触发可能被封IP或冻结Cookie几小时。
- **不要硬编码任何网盘API Key**：所有凭证走用户配置。
- **不要存储用户网盘密码**：仅存储登录后的持久化Cookie。密码泄露会影响用户其他服务安全。

### 1.4 安全红线
- **激活码/授权密钥绝对不能硬编码在客户端**：必须通过Cloudflare Workers验证，客户端只传key不做本地校验。
- **VIP进阶代码绝不打包进基础版安装包**：必须从Cloudflare R2云端拉取，且每次拉取附带license校验。
- **GitHub公开仓库不得包含**：网盘API密钥、Cloudflare Admin Token、数据库连接字符串、完整的兑换码生成逻辑。

### 1.5 GUI设计
- **基础版UI不得出现灰色不可用按钮**：VIP功能要么完全不显示，要么通过VIP入口引导，不能放灰按钮让用户看着难受。
- **不得在日志或错误提示中暴露内部路径**：避免泄露用户目录结构（如 C:\\Users\\xxx）。

## 二、已知坑点 (GOTCHAS)

### 2.1 夸克网盘
- Cookie 失效快：部分用户的夸克Cookie会在数小时内过期。需要做「Cookie有效性检测 + 失效弹窗提醒 + 引导重新登录」。
- 转存需要目标目录ID：夸克转存API需要用户网盘的目标文件夹ID，不能直接用路径。
- 大文件下载可能限速：建议预留限速场景，给用户显示「当前速度」而非只显示百分比。

### 2.2 MTP / DBI
- Switch进入DBI模式后 Windows 会把它识别为便携设备，不是常规驱动器。PyMTP 的 `detect_devices()` 需要过滤出 Switch（通过VID/PID或设备名）。
- DBI第5分区 = TF卡、第6分区 = NAND：这个映射在不同DBI版本中可能不同，需要实际测试确认。
- 传输完文件后 DBI 不会立即开始安装：可能需要等文件全部传完 + 用户手动在Switch上点安装，或者在DBI设置中开启自动安装模式。

### 2.3 Playwright 爬虫
- nsthwj.cn 可能检测 headless 浏览器：需要配置反检测参数（如伪装 webdriver 属性、userDataDir持久化）。
- 网站结构可能随时变化：爬虫需要做好异常处理，页面元素找不到时应返回明确错误而非崩溃。
- Cloudflare 验证：如果网站加了 Cloudflare Turnstile 验证，Playwright 可能无法绕过，需要降级为手动提示用户操作。

### 2.4 Windows 打包
- PyInstaller 打包 PyMTP 时，libmtp 的 DLL（libusb、libmtp-9.dll）需要手动包含在打包配置中。
- Electron 启动 Python 子进程时，`__dirname` 在打包后会变，`python.exe` 的路径需要正确处理。
- Windows Defender 可能误报 PyInstaller 打包的 exe 为病毒。

## 三、兼容性约束

### 3.1 操作系统
- PC端：Windows 10 / 11（不保证 Windows 7/8 兼容）
- macOS：暂不支持
- Switch：大气层 1.6.0+，系统固件 17.0+

### 3.2 Switch 固件版本
- DBI 需要较新的 MTP 实现，建议用户升级到 DBI 658+ 版本
- 大气层升级功能仅支持 Atmosphere 1.6+ 版本

### 3.3 网盘
- 夸克网盘：需用户有关联的超会或SVIP才能满速下载（普通用户限量）
- 百度网盘：需 SVIP
- 阿里云盘：对免费用户相对友好

## 四、代码规范

### 4.1 Python
- 遵循 PEP 8
- 类型注解：所有函数参数和返回值必须有类型注解
- 异步优先：IO密集型操作（HTTP请求、文件读写）一律使用 async/await
- 日志：使用 Python logging 模块，禁止 print()

### 4.2 TypeScript / React
- 严格模式：tsconfig 开启 strict: true
- 函数组件 + Hooks：不使用 class component
- CSS：统一使用 Tailwind utility class，不写自定义 CSS 文件（除非特殊动画效果）

### 4.3 通用
- 所有用户可见文本走 i18n 常量（初期中文，预留英文扩展）
- 错误信息必须对用户友好，不含技术栈信息
