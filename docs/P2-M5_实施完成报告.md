## ⚠️ 注意事项

### 1. pywin32/comtypes 打包

- PyInstaller 已配置相应的 hidden imports
- Hook 文件会自动收集依赖
- 如果遇到 COM 组件错误，检查 `hook-comtypes.py`

### 2. Playwright 浏览器

- 当前配置：首次启动时下载（不打包）
- 如需打包：在 `build.spec` 中取消注释 `datas` 配置
- 体积会增加约 150MB

### 3. 文件大小

- 预计 Python 后端：50-100MB（含依赖）
- 预计安装包：80-150MB（不含浏览器）
- 启用 UPX 压缩可减小约 30%

### 4. 权限要求

- MTP 操作可能需要管理员权限
- 安装程序支持 UAC 提升
- 默认安装到 `C:\Program Files`
