## 一、Python核心服务模块划分

### 1.2 关键模块说明

**scraper (爬虫)**
- `client.py`：封装 Playwright 异步浏览器，管理 Cookie 持久化
- `parser.py`：DOM 选择器提取游戏标题、版本号、网盘跳转链接
- `game_dict.py`：维护「别名→标准名」映射，支持模糊搜索匹配

**cloud_disk (网盘)**
- `base.py`：定义统一接口 `save_to_drive(url)` / `get_download_link(file_id)` / `download(file_id, dest, progress_callback)`
- 各实现类只需覆写差异部分，方便扩展新网盘

**mtp (MTP传输)**
- `device.py`：通过 PyMTP 扫描设备，识别 DBI 模式下的 Switch
- `transfer.py`：单文件串行传输，每 1MB 回调一次进度
- `dbi_layout.py`：判断第5分区(TF卡)/第6分区(主机内存)状态，自动选择分区

## 二、本地 HTTP API 接口清单

### 2.1 游戏搜索

| 方法 | 路径 | 参数 | 返回 |
|---|---|---|---|
| GET | `/api/search` | `?keyword=塞尔达` | 搜索结果列表 [{title, version, size, source_url}] |
| GET | `/api/game/detail` | `?url=...` | 详情 {title, body_url, update_url, dlc_url, cheat_url} |

### 2.2 安装管理

| 方法 | 路径 | 参数 | 返回 |
|---|---|---|---|
| POST | `/api/install` | Body: `{game_url, install_order}` | 任务ID，启动异步安装 |
| GET | `/api/install/{task_id}/progress` | | {stage, percent, current_file, speed} |
| POST | `/api/install/local` | Body: `{folder_path}` | 任务ID，安装本地已下载游戏 |
| POST | `/api/install/batch` | Body: `{game_list: [...]}` | 批量安装任务ID（VIP） |

### 2.3 设备检测

| 方法 | 路径 | 返回 |
|---|---|---|
| GET | `/api/device/switch` | {connected: bool, mode: "DBI", free_space_tf: 0, free_space_nand: 0} |
| GET | `/api/device/tfcard` | {inserted: bool, drive_letter: "E:", free_space: 0} |

### 2.4 设置

| 方法 | 路径 | 参数 |
|---|---|---|
| GET | `/api/settings` | 返回当前配置 |
| PUT | `/api/settings` | Body: `{key: value}` |
| POST | `/api/auth/activate` | Body: `{code: "XXXX-XXXX-XXXX"}` |

### 2.5 系统

| 方法 | 路径 | 返回 |
|---|---|---|
| GET | `/api/version` | {current: "1.0.0", latest: "1.1.0", update_url: ...} |
| GET | `/api/health` | {status: "ok", python_version: "3.11", uptime: 3600} |

## 四、D1 数据库表设计

```sql
-- 兑换码表
CREATE TABLE activation_codes (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    code        TEXT NOT NULL UNIQUE,       -- 兑换码，格式: YYZZ-XXXX-XXXX
    batch_id    TEXT,                       -- 批次标识
    valid_days  INTEGER NOT NULL DEFAULT 365, -- 有效期天数
    max_devices INTEGER NOT NULL DEFAULT 2, -- 最大绑定设备数
    is_used     INTEGER NOT NULL DEFAULT 0, -- 是否已使用
    created_at  TEXT NOT NULL DEFAULT (datetime('now')),
    used_at     TEXT
);

-- 授权记录表
CREATE TABLE licenses (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    license_key     TEXT NOT NULL UNIQUE,   -- 授权密钥 UUID
    code_id         INTEGER NOT NULL REFERENCES activation_codes(id),
    device_id       TEXT NOT NULL,          -- 设备唯一ID
    device_type     TEXT NOT NULL DEFAULT 'pc', -- pc / switch
    activated_at    TEXT NOT NULL DEFAULT (datetime('now')),
    expires_at      TEXT NOT NULL,          -- 到期时间
    last_heartbeat  TEXT,
    is_revoked      INTEGER NOT NULL DEFAULT 0,
    UNIQUE(code_id, device_id)
);

-- 管理后台用户表
CREATE TABLE admin_users (
    id       INTEGER PRIMARY KEY AUTOINCREMENT,
    username TEXT NOT NULL UNIQUE,
    password_hash TEXT NOT NULL,
    is_super INTEGER NOT NULL DEFAULT 0
);

-- 操作审计日志
CREATE TABLE audit_log (
    id         INTEGER PRIMARY KEY AUTOINCREMENT,
    action     TEXT NOT NULL,
    detail     TEXT,
    ip         TEXT,
    created_at TEXT NOT NULL DEFAULT (datetime('now'))
);
```

## 五、核心数据流

### 5.1 单游戏安装流程

```
用户输入游戏名
  │
  ▼
[Electron GUI] ──GET /api/search?keyword=xxx──▶ [Python scraper]
                                                    │
                                                    ▼
                                            Playwright 访问 nsthwj.cn
                                            搜索 → 解析结果列表
                                            返回 {title, url, version, ...}
                                                    │
  ◀───────────── 搜索结果列表 ───────────────────────┘
  │
  ▼ 用户确认选择
[Electron GUI] ──POST /api/install {game_url}──▶ [Python install orchestrator]
                                                    │
                                          ┌─────────┼─────────┐
                                          ▼         ▼         ▼
                                      [scraper] [cloud_disk] [cache]
                                      抓取详情页  夸克转存    检查缓存
                                      获取网盘链接  获取直链
                                          │         │         │
                                          └─────────┼─────────┘
                                                    ▼
                                              下载到本地缓存
                                              (%APPDATA%/youyuzz/cache/)
                                                    │
                                                    ▼
                                          [game_files classifier]
                                          分类：本体 / 更新 / DLC / 金手指
                                          按顺序排列安装队列
                                                    │
                                                    ▼
                                          [mtp transfer]
                                          检测Switch DBI连接
                                          选择分区(TF卡优先)
                                          逐个文件 MTP 传输
                                          每1MB回调进度
                                                    │
  ◀─────── SSE/轮询 实时进度 ────────────────────────┘
  │
  ▼
[Electron GUI] 显示安装完成
[PushDeer] 推送通知（VIP）
```
