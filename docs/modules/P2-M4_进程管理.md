# P2-M4: 进程管理

> pc-client/frontend/electron/ | 待开始

## 目标
Electron 主进程中管理 Python 后端的完整生命周期。

## 功能
- 启动时自动拉起 Python FastAPI 子进程
- 健康检查：轮询 /api/health 直到就绪
- 异常恢复：Python 崩溃时自动重启
- 关闭时优雅退出 Python 子进程
- stdout/stderr 日志捕获

## 关键文件
`
electron/
├── main.ts           # 窗口创建 + Python 进程管理
├── pythonBridge.ts   # Python 子进程封装
└── preload.ts        # 预加载脚本
`
