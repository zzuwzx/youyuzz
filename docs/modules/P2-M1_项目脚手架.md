# P2-M1: Electron + React 脚手架

> pc-client/frontend/ | ✅ 完成

## 目标
搭建 Electron + Vite + React + TypeScript + Tailwind CSS 项目骨架。

## 技术栈
- Electron 30+ — 桌面壳
- Vite 5 — 构建工具
- React 18 + TypeScript 5 — UI 框架
- Tailwind CSS 3 — 原子化样式

## 文件结构
`
frontend/
├── package.json
├── tsconfig.json
├── vite.config.ts
├── tailwind.config.js
├── postcss.config.js
├── index.html
├── electron/
│   └── main.ts        # Electron 主进程
├── src/
│   ├── main.tsx        # React 入口
│   ├── App.tsx         # 根组件 + 路由
│   ├── index.css       # Tailwind 入口
│   ├── components/     # UI 组件
│   ├── pages/          # 页面
│   └── hooks/          # 自定义 Hooks
`
