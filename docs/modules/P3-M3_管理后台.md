# P3-M3: Cloudflare Pages 管理后台

> cloud/admin/ | 待开始

## 目标
网页端管理后台，生成兑换码、查看用户、统计概览。

## 页面
| 页面 | 路由 | 内容 |
|---|---|---|
| 登录 | /login | 管理员用户名+密码 |
| 兑换码管理 | /codes | 批量生成、查看所有码 |
| 用户列表 | /users | 激活用户、到期时间、状态 |
| 统计概览 | /stats | 总用户、活跃、今日新增 |

## 技术栈
- 静态 HTML + Vanilla JS（或轻量框架）
- 调用 Workers API（同域）
