# P2-M5: 打包发布

> pc-client/frontend/ | 待开始

## 目标
打包为 Windows 安装包（.exe），Python 后端嵌入安装。

## 打包方案
- Python 后端：PyInstaller → single .exe（含 libmtp DLL）
- Electron 前端：electron-builder → NSIS 安装包
- 最终产出：setup.exe，用户一键安装

## electron-builder 配置（package.json build 段已有基础）
- extraResources: Python exe 嵌入安装目录
- NSIS: 允许自定义安装路径
