> pc-client/frontend/ | ✅ 核心功能已完成 (78%)

## 使用方法

### 运行 Electron 应用
```powershell
pc-client\frontend\release\win-unpacked\鱿郁仔仔.exe
```

### 运行 Python 后端 (独立)
```powershell
pc-client\backend\dist\backend.exe
```

### 重新打包
```powershell
cd scripts
.\build.ps1
```

### 验证安装包
```powershell
cd scripts
.\verify-installer.ps1 -InstallerPath "path\to\installer.exe"
```
