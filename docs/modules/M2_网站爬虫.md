# M2: 网站爬虫模块

> pc-client/backend/scraper/ | 待开发

## 目标
访问 nsthwj.cn，搜索游戏→解析结果→获取详情页和网盘链接。

## 技术方案
- Playwright异步浏览器(反headless检测)
- Cookie持久化存储
- 游戏名模糊匹配词典

## 文件结构
`
scraper/
├── client.py     # Playwright封装,Cookie管理
├── parser.py     # DOM解析
└── game_dict.py  # 模糊匹配词典
`
