# M2: 网站爬虫模块

> pc-client/backend/scraper/ | ✅ 已完成

## 目标
访问 nsthwj.cn，搜索游戏→解析结果→获取详情页和网盘链接。

## 技术方案
- Playwright异步浏览器(反headless检测)
- Cookie持久化存储
- 游戏名模糊匹配词典

## 文件结构
`
scraper/
├── client.py     # Playwright封装,Cookie管理
├── parser.py     # DOM解析
└── game_dict.py  # 模糊匹配词典
`

### 异常处理覆盖
| 场景 | 策略 |
|---|---|
| 网络超时/浏览器崩溃 | search() 外层 3 次重试 × 2s 间隔 |
| Cloudflare 挑战 | DETECT_CF_JS 检测 title + #challenge-running，命中返回空 + warning |
| Vue 结构变化 | 搜索框超时 5s → 返回空 + error；提取空数组 → 返回空 + warning（附带关键词） |
| 搜索框定位失败 | locator.fill() 包 try/except，selector 失效时降级返回空 |
