# M4: 游戏文件识别模块

> pc-client/backend/game_files/ | 有现成代码移植

## 目标
扫描本地游戏资源目录，自动分类本体/更新/DLC/金手指，按安装顺序排列。

## 技术方案
从SquidInstallerGUI.py的SmartScanner.scan_folder()移植，返回Pydantic模型。

## 文件结构
`
game_files/
├── classifier.py  # 文件分类(移植SmartScanner)
└── cheat.py       # 金手指压缩包处理
`

## 现成代码
SquidInstallerGUI.py SmartScanner已实现:
- .nsp/.nsz/.xci检测
- 关键词分类: upd/update→更新, dlc/追加内容→DLC
- .zip金手指检测
- priority排序(本体→更新→DLC)
