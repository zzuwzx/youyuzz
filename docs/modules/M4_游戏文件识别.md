# M4: 游戏文件识别模块

> pc-client/backend/game_files/ | ✅ 已完成移植 + v1.1 增强 | 33/33 tests pass

## 目标
扫描本地游戏资源目录，自动分类本体/更新/DLC/金手指，按安装顺序排列。

## 技术方案
从 SquidInstallerGUI.py 的 SmartScanner.scan_folder() 移植，返回 Pydantic 模型。

## 文件结构
`
game_files/
├── classifier.py  # 文件分类（SmartScanner + 正则版本检测）
├── cheat.py       # 金手指压缩包处理（校验/解压）
├── models.py      # Pydantic 数据模型
└── __init__.py    # 模块导出
`

## 核心功能

### 文件格式支持
- .nsp / .nsz / .xci / .xcz 游戏文件
- .zip 金手指压缩包

### 分类规则
1. 优先检测更新（显式关键词 OR 正则版本号检测）
2. 其次检测 DLC（关键词匹配）
3. 再次检测本体（关键词匹配）
4. 无明确标识 → 默认本体

### 版本号检测 (_is_update_version)
- v65536 / v131072 / v196608 / v262144 → 更新（Switch 标题版本 ID）
- vX.Y.Z 且 patch > 0 或 minor > 0 → 更新
- vX.Y 且 Y > 0 → 更新
- v1.0 / v2.0 / v1.0.0 → 本体
