# M3: 网盘集成模块

> pc-client/backend/cloud_disk/ | 待开发

## 目标
夸克网盘: Cookie登录→文件转存→下载直链→断点下载。
后续扩展百度/阿里(Phase 4 VIP)。

## 技术方案
- 抽象基类CloudDiskBase: save_to_drive/get_download_link/download
- 夸克实现: httpx+Cookie jar
- 下载进度: httpx.stream + num_bytes_downloaded

## 文件结构
`
cloud_disk/
├── base.py     # 抽象基类
├── kuake.py    # 夸克网盘
├── baidu.py    # 百度(Phase4)
└── aliyun.py   # 阿里(Phase4)
`
