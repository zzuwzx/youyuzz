# M3: 网盘集成模块

> pc-client/backend/cloud_disk/ | ✅ 完成 | 2026-05-25

## 目标
夸克 / 百度 / 阿里云盘：扫码登录 → 文件转存 → 下载直链 → 多分段全速下载。

## 技术方案
- 抽象基类 `CloudDiskBase`：`login_via_browser()` / `save_to_drive` / `get_download_link` / `download(segments=N)`
- **扫码登录**：`login_via_browser()` — Playwright 弹出浏览器 → 手机扫码 → 自动抓取 Cookie（夸克检测 `__pus`，百度 `BDUSS`，阿里 `refresh_token`）
- **夸克实现**：httpx + Cookie jar + share token → save API；默认 4 分段下载
- **百度实现**：BDUSS/STOKEN + HTML 解析 + MD5 sign + transfer API；默认 8 分段突破限速
- **阿里云实现**：refresh_token → access_token 自动刷新 + batch copy API；默认 4 分段下载
- **多分段并行下载**：`download(segments=N)` — HTTP Range 分片 → asyncio.gather 并发 → 合并 .part 文件；不支持 Range 则自动降级单段
- **下载进度**：httpx.stream + 汇总进度回调（500ms throttle）
