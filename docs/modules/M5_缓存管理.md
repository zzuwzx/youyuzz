# M5: 缓存管理模块

> pc-client/backend/cache/ | ✅ 已完成

## 目标
下载后本地缓存1天TTL，再次安装同游戏优先读缓存。

## 技术方案
- 缓存目录: %APPDATA%/youyuzz/cache/
- TTL: 基于文件修改时间
- 空间监控: 低于阈值自动清理最旧缓存

## 文件结构
`
cache/
├── manager.py   # 缓存读写,TTL管理
└── storage.py   # 磁盘空间监控
`
