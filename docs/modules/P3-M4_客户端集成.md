# P3-M4: 客户端授权集成

> pc-client/ | ✅ 已完成（2026-05-28）

## 目标
客户端启动时云端验证，激活码输入，VIP功能开关。

## 功能
1. 启动时 POST /api/auth/verify → 验证 license_key
2. 激活码输入对话框 → POST /api/auth/activate
3. 基础版/VIP功能开关（根据 license type）
4. 授权到期提醒弹窗
5. 心跳上报（每小时 POST /api/auth/heartbeat）
6. 防破解：每次操作前校验，不本地缓存授权
