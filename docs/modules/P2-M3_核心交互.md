## 实现详情

### 1. API 通信层
- **文件**: `src/api/client.ts`
- **功能**: 
  - 统一的 fetch wrapper
  - 基础 URL: `/api` (通过 Vite 代理到 localhost:18888)
  - 自动 JSON 解析
  - 统一错误处理
  - TypeScript 类型支持

### 2. 搜索交互
- **组件**: `SearchBox.tsx`, `GameList.tsx`
- **Hook**: `useSearch.ts`
- **功能**:
  - 输入关键词触发搜索
  - 显示搜索结果列表
  - 选择游戏进行安装
  - Loading/Empty/Error 状态

### 3. 安装交互
- **组件**: `InstallProgress.tsx`
- **Hook**: `useInstall.ts`
- **功能**:
  - POST /api/install 启动安装
  - 每秒轮询 /api/install/{id}/progress
  - 实时更新进度条和状态
  - 显示速度、ETA、当前文件
  - 支持取消安装

### 4. 设备检测
- **组件**: `DeviceStatus.tsx`, `Modal.tsx`
- **Hook**: `useDevice.ts`
- **功能**:
  - 启动时自动检测 Switch 连接
  - 显示设备状态（连接、DBI模式、TF卡）
  - 未连接时弹窗提醒
  - 支持手动重新检测

### 5. 本地离线安装
- **组件**: `LocalInstall.tsx`
- **功能**:
  - 文件夹选择对话框
  - 支持手动输入路径
  - POST /api/install/local 调用

### 6. 设置持久化
- **组件**: `Settings.tsx`
- **Hook**: `useSettings.ts`
- **功能**:
  - GET /api/settings 加载配置
  - PUT /api/settings 保存配置
  - 支持下载路径、自动安装、通知等设置

### 7. 状态组件
- **Loading**: 加载中动画
- **Error**: 错误提示 + 重试按钮
- **Empty**: 空状态提示

## 使用说明

1. **搜索游戏**: 在搜索框输入关键词，点击搜索
2. **选择游戏**: 点击游戏列表中的游戏进行选择
3. **开始安装**: 选中游戏后，点击"开始安装"按钮
4. **查看进度**: 在右侧面板查看安装进度
5. **本地安装**: 点击"本地安装"按钮，选择游戏文件夹
6. **设备检测**: 启动时自动检测，也可手动点击"检测设备"
7. **设置**: 点击左上角菜单进入设置页面

---

## 注意事项

- 需要先连接 Switch 设备才能安装游戏
- 确保 Switch 已开启 DBI 模式
- TF 卡需要正确插入
- 安装过程中请勿断开设备连接
- Vite 代理配置: `/api` → `http://localhost:18888`
