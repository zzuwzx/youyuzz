# P2-M3: 核心交互

> pc-client/frontend/src/ | 待开始

## 目标
通过 localhost:18888 API 与 Python 后端通信，完成搜索→安装全流程。

## API 调用清单
| 功能 | 方法 | 路径 |
|---|---|---|
| 游戏搜索 | GET | /api/search?keyword= |
| 游戏详情 | GET | /api/game/detail?url= |
| 开始安装 | POST | /api/install |
| 安装进度 | GET | /api/install/{id}/progress |
| 本地安装 | POST | /api/install/local |
| Switch状态 | GET | /api/device/switch |
| TF卡状态 | GET | /api/device/tfcard |
| 读取配置 | GET | /api/settings |
| 保存配置 | PUT | /api/settings |
| 健康检查 | GET | /api/health |

## 设备检测
- 启动时自动检测 Switch 连接状态
- 未连接弹窗提醒 + 重新检测按钮
- TF 卡未插入弹窗提醒

## 本地离线安装
- 文件夹选择对话框 → 调后端 API → MTP 传输
