# P3-M1: D1 数据库 Schema

> cloud/worker/ | ✅ 已完成（2026-05-28）

## 目标
设计并创建 D1 数据库表结构。

## 表清单（来自 docs/02_系统设计文档.md）

### activation_codes
| 字段 | 类型 | 说明 |
|---|---|---|
| id | INTEGER PK AUTO | 主键 |
| code | TEXT UNIQUE | 兑换码 YYZZ-XXXX-XXXX |
| batch_id | TEXT | 批次标识 |
| valid_days | INTEGER DEFAULT 365 | 有效期天数 |
| max_devices | INTEGER DEFAULT 2 | 最大绑定设备数 |
| is_used | INTEGER DEFAULT 0 | 是否已使用 |
| created_at | TEXT DEFAULT datetime(now) | 创建时间 |
| used_at | TEXT | 使用时间 |

### licenses
| 字段 | 类型 | 说明 |
|---|---|---|
| id | INTEGER PK AUTO | 主键 |
| license_key | TEXT UNIQUE | UUID 授权密钥 |
| code_id | INTEGER FK | 关联 activation_codes |
| device_id | TEXT | 设备唯一ID |
| device_type | TEXT DEFAULT 'pc' | pc / switch |
| activated_at | TEXT DEFAULT datetime(now) | 激活时间 |
| expires_at | TEXT | 到期时间 |
| last_heartbeat | TEXT | 最后心跳 |
| is_revoked | INTEGER DEFAULT 0 | 是否吊销 |

### admin_users
| 字段 | 类型 | 说明 |
|---|---|---|
| id | INTEGER PK AUTO | 主键 |
| username | TEXT UNIQUE | 用户名 |
| password_hash | TEXT | 密码哈希 |
| is_super | INTEGER DEFAULT 0 | 是否超级管理员 |

### audit_log
| 字段 | 类型 | 说明 |
|---|---|---|
| id | INTEGER PK AUTO | 主键 |
| action | TEXT | 操作类型 |
| detail | TEXT | 操作详情 |
| ip | TEXT | 来源IP |
| created_at | TEXT DEFAULT datetime(now) | 时间 |
